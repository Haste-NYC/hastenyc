     - Uses useEffect to set up IntersectionObserver

     - Clean up observer on unmount
     - Return activeSection string

     - When active: after:scale-x-100
   - Wrap the nav link in relative class to position the underline
   - Apply same active styling to mobile menu links

  <name>Task 2: Add hero parallax and enhance accordion/pricing animations</name>
  <files>src/components/HeroSection.tsx, src/components/PricingPlans.tsx, tailwind.config.ts</files>
  <action>
1. Update src/components/HeroSection.tsx for parallax:
   - Import useRef from 'react'
   - Import useScroll, useTransform from 'framer-motion'
   - Import useMediaQuery hook or create inline: const isDesktop = typeof window !== 'undefined' && window.matchMedia('(min-width: 768px)').matches
     - Better: Use state + effect to check media query on client
   - Add ref to the section element: const sectionRef = useRef(null)
   - Set up scroll tracking:
     ```
     const { scrollYProgress } = useScroll({
       target: sectionRef,
       offset: ["start start", "end start"]
     });
     ```
   - Create parallax transform (desktop only):
     ```
     const backgroundY = useTransform(
       scrollYProgress,
       [0, 1],
       ['0%', '30%']
     );
     ```
   - Apply to gradient background div:
     - Wrap gradient div in motion.div
     - Add style={{ y: isDesktop ? backgroundY : 0 }}
     - Keep pointer-events-none on the gradient
   - Add ref={sectionRef} to the section element

2. Update src/components/PricingPlans.tsx for hover:
   - Import motion from 'framer-motion'
   - Wrap each Card in motion.div with:
     ```
     <motion.div
       whileHover={{ y: -4, transition: { duration: 0.2 } }}
       className="h-full"
     >
       <Card ...>
     </motion.div>
     ```
   - Keep existing Tailwind hover classes (transition-all, hover:border-*) on Card
   - The motion.div handles the lift, Card handles border/shadow
   - Add hover:shadow-lg hover:shadow-purple-500/10 to Card className for glow effect

3. Update tailwind.config.ts for smoother accordion:
   - Find the accordion-down keyframes
   - Add opacity to the animation:
     ```
     "accordion-down": {
       from: { height: "0", opacity: "0" },
       to: { height: "var(--radix-accordion-content-height)", opacity: "1" },
     },
     "accordion-up": {
       from: { height: "var(--radix-accordion-content-height)", opacity: "1" },
       to: { height: "0", opacity: "0" },
     },
     ```
   - Update animation timing:
     ```
     animation: {
       "accordion-down": "accordion-down 0.3s cubic-bezier(0.87, 0, 0.13, 1)",
       "accordion-up": "accordion-up 0.3s cubic-bezier(0.87, 0, 0.13, 1)",
     },
     ```
   - The cubic-bezier is "ease-in-out-expo" - snappy start and end, smooth middle

IMPORTANT for parallax: Only apply parallax on desktop (md breakpoint and up). Mobile should have static background.
  </action>
