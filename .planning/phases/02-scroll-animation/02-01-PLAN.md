  <action>
1. Install Lenis:
   ```bash
   npm install lenis
   ```

2. Create src/components/SmoothScrollProvider.tsx:
   - Import ReactLenis from 'lenis/react'
   - Import MotionConfig from 'framer-motion'
   - Create SmoothScrollProvider component that wraps children with:
     - MotionConfig with reducedMotion="user" (outer wrapper)
     - ReactLenis with options: lerp: 0.1, duration: 1.2, smoothWheel: true, syncTouch: false
   - Export the component

Configuration rationale:
- lerp: 0.1 gives buttery smooth interpolation
- duration: 1.2 for programmatic scrolls (scrollTo calls)
- smoothWheel: true for mouse wheel smoothing
- syncTouch: false to avoid iOS instability (per research)
- reducedMotion: "user" respects OS preference automatically

  <action>
1. Update src/App.tsx:
   - Import SmoothScrollProvider from "@/components/SmoothScrollProvider"
   - Wrap the entire app content (inside QueryClientProvider but outside BrowserRouter) with SmoothScrollProvider
   - Keep existing ScrollToTop component (Lenis handles scroll-to-top differently but this is for route changes)

2. Update src/components/Header.tsx:
   - Import useLenis from 'lenis/react'
   - Get lenis instance: const lenis = useLenis();
   - Replace the scrollToSection function body:
     - REMOVE: window.scrollTo({ top: offsetPosition, behavior: 'smooth' })
     - ADD: lenis?.scrollTo(`#${id}`, { offset: -80, duration: 1.2, easing: (t) => Math.min(1, 1.001 - Math.pow(2, -10 * t)) })
   - The offset: -80 accounts for fixed header (80px)
   - The easing function is exponential ease-out for premium feel

3. Update src/components/HeroSection.tsx:
   - Import useLenis from 'lenis/react'
   - Get lenis instance at top of component
   - Replace scrollToPricing function body:

   - Update the "Watch Demo" onClick handler similarly:
     - REMOVE: window.scrollTo({ top: offsetPosition, behavior: "smooth" })
     - ADD: lenis?.scrollTo('#video', { offset: -80, duration: 1.2 })

IMPORTANT: Remove all window.scrollTo calls from these files. Lenis must handle ALL scrolling for consistent behavior.
